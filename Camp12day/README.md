## GridView
`android:columnCount="3"`

가로축 기준으로 3칸씩 나누겠다.


`android:layout_columnSpan="2"`

`android:layout_columnWeight="1"`

가로에 몇칸을 차지할것인지 설정


`android:layout_row="3"`

`android:layout_column="0"`
가로 세로 지정된 칸에 배치 가능
